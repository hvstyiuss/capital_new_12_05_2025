Just check:

✓ Controllers

* Are requests validated?

* Is authorization applied?

* Are model queries safe?

✓ Models

* Are relationships correct?

* Are fillable fields secure?

✓ Migrations

* Are columns correctly typed?

* Are foreign keys set properly?

✓ Routes

* Are routes protected?

* No sensitive endpoints left public?

✓ Security

* User data validation

* Avoiding mass assignment

* Using policies or guards

✓ Logic

* Clean structure

* No duplicated code

* No business logic inside controllers
